# Настройка API ключей

## GraphHopper API

Ключ GraphHopper уже добавлен в проект: `aa902198-c697-4891-a0f0-6a443a3e8889`

### Добавление ключа в переменные окружения

Создайте файл `.env.local` в корне проекта `parkovka-app/` и добавьте:

```env
# GraphHopper API (для маршрутизации)
GRAPHHOPPER_API_KEY=aa902198-c697-4891-a0f0-6a443a3e8889
NEXT_PUBLIC_GRAPHHOPPER_API_KEY=aa902198-c697-4891-a0f0-6a443a3e8889
```

### Как работает маршрутизация

Система пробует провайдеры в следующем порядке:

1. **Yandex Maps API** (если есть ключ) - лучший для России
2. **GraphHopper API** (если есть ключ) - хорошая альтернатива
3. **OSRM** (бесплатный, не требует ключа) - fallback

### Использование

Ключ GraphHopper будет автоматически использоваться для построения маршрутов, если:
- Yandex Maps API ключ не настроен или недоступен
- GraphHopper API доступен и работает

## Yandex Maps API (опционально)

Для геокодирования адресов и лучшей маршрутизации в России рекомендуется также добавить Yandex Maps API ключ:

```env
# Yandex Maps API (для геокодирования и маршрутизации)
YANDEX_MAPS_API_KEY=your_yandex_key_here
NEXT_PUBLIC_YANDEX_MAPS_API_KEY=your_yandex_key_here
```

Получить ключ можно на: https://developer.tech.yandex.ru/

## Проверка работы

После добавления ключей перезапустите сервер разработки:

```bash
npm run dev
```

Маршрутизация будет работать автоматически при создании бронирований.
